# Used Machinery Intelligence Platform

Sales-decision tool for used sheet metal / structural steel fabrication machinery: customer requirement matching, fair market valuation, and a Pricing & Profitability module for live quoting.

**Status:** Design phase. See [ARCHITECTURE.md](./ARCHITECTURE.md) for the full system architecture, database schema, scraping strategy, API design, AI workflow, and the Pricing & Profitability module spec. Open decisions requiring sign-off are listed at the end of that document — MVP implementation starts once those are confirmed.
